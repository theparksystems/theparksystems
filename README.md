## Railway

Railway can deploy this repository directly from GitHub.

The app uses:

```bash
pnpm install
pnpm run build
pnpm start
```

`pnpm start` launches `server.mjs`, which reads Railway's `PORT` environment
variable, defaults to port `8080`, and binds to `0.0.0.0`.

## Local preview

```bash
pnpm install
pnpm run dev
```

## Production build

```bash
pnpm run build
pnpm start
```
